# JFPI (Just Play It)

(Rapid Prototyping, Minimum Valuable Product, & Iterative Development) crossed with (Tutorial Missions, M:tG Portal, & Expansions)

This repository reduces the ruleset of some of my game collection to the bare minimum needed for people to play a version of that game.
The idea being that the core game idea should be playable with under 2 minutes of explanation & setup.
If they like the game they can then add in other mechanics iteratively.
If they don't like the game they can find out quickly and move on.

There should be no part of this repository that is a complete game, if you find such an occasion please raise an issue or submit a pull request to remove it.
Wherever possible contributors should direct players to read specific sections of the ruleset rather than replicate them in this repository.
